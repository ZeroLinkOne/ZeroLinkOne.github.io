## 回归Github

好久没更新，重新回归 GitHub 博客，却发现连 Git 都快忘了怎么用。索性把操作流程完整记录下来，留作备忘。

## 下载Git for windows

	下载[Git for windows],地址https://gitforwindows.org, 双击安装，一路Next安装成功。

	下载[汉化包],地址https://github.com/stayor/git-gui-zh,复制汉化包到 Git\mingw64\share\git-gui\lib\msgs。
	
## 创建ssh key

    Git Bash
    
    cd ~/.ssh
    
    ssh-keygen -t rsa -C "email@example.com"
    
    cat id_rsa.pub 查看生成的密钥
    
    登录github -> settings -> SSH and GPG keys -> New SSH key -> 输入标题和密钥
    
    ssh -T git@github.com 查看是否成功

## 创建ssh config

    Git Bash
    
    git config --global user.email "email@example.com"

## 创建博客

    登录Github,创建仓库 username.github.io
    
    增加一个readme.md
    
    git clone git@github.com:username/username.github.io
    
    复制Jekyll样式到博客文件夹
    
    GitGUI -> Stage Changed -> commit -> push -> 输入密码 -> Ok
    
    等待一会就可以查看博客内容了

## Tips

    原有的ssh key可删除， rm -rf ~/ssh/*
